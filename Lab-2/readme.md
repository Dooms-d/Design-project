## Project Overview

This lab focuses on documenting the components of a web application using Information Architecture (IA) techniques. The documentation includes:

1. **Site Map** - Visual representation of the website's page hierarchy and navigation structure
2. **Asset Inventory** - Comprehensive spreadsheet cataloging all pages and their associated assets

## Site Structure

The Soilboy website consists of 6 main pages:

```
1.0 Home (index)
    |
    +-- 2.0 Plants
    |       +-- 2.1 Potted Plants
    |       +-- 2.2 Planters
    |       +-- 2.3 Essentials
    |       +-- 2.X Plants (individual product pages)
    |
    +-- 3.0 Workshops
    |
    +-- 4.0 Lifestyle
    |       +-- 3.X Lifestyle Products
    |
    +-- 5.0 Product Details (dynamic template)
    |
    +-- 6.0 Contact
```
