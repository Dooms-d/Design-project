## Mission Statement

Paw & Whiskers is a volunteer organization dedicated to empowering a community that champions stray cats. We provide care and forever homes for stray felines while funding our efforts through a curated selection of cat-related products. Every purchase makes a pawsitive impact, ensuring a brighter future for feline friends in need.

**Tagline:** "Shop with Purpose, Love with All Your Heart" | "Nurturing Strays, Enriching Life"

  - Home

  - Contact Us
- **Call-to-Action:** "BUY NOW" button

#### Footer Design
- **Layout:** Multi-column footer
- **Color Scheme:** #206A5D (brand color)
- **Elements:**
  - Copyright information (centered, white text)
  - Quick links to main pages
  - Social media integration (prepared for icons)
- **Customization:** Used Astra Footer Builder

### Content Blocks Used

#### Home Page Blocks:
1. **Cover Block**
   - Hero image with text overlay
   - Minimum height: 600px
   - Background image: Rescue cat photography
   - Text alignment: Left-aligned for readability

2. **Columns Block**
   - Two-column layout for featured content
   - Left: Icon/Image (50x50px, centered)
   - Right: Text paragraphs
   - Vertical alignment: Middle
   - Typography: Medium size headings

3. **Group Block**
   - Background color: #F1F1E8 (soft, welcoming tone)
   - Contains nested blocks for products section
   - Padding and spacing optimized for readability

4. **Image Gallery**
   - Responsive 3-column grid
   - Rounded corners for modern aesthetic
   - Showcases "Furry Buddies"

5. **Post Grid Block (Spectra)**
   - Displays product/blog posts dynamically
   - Card-based layout
   - Featured images for each product
   - "Discover to Support" section

#### Blog Page Features:
- **Post Grid Layout:** Automated post aggregation
- **Featured Images:** Required for all products
- **Categories/Tags:** Organized by product type
- **Excerpts:** Short descriptions on grid view

## WordPress Blocks & Page Building Concepts

### 1. **Block Editor Fundamentals**
- **What are Blocks?** Modular content elements that can be added, rearranged, and styled
- **Block Types Used:**
  - Layout blocks (Columns, Group, Cover)
  - Media blocks (Image, Gallery)
  - Text blocks (Paragraph, Heading)
  - Widget blocks (Post Grid)

### 2. **Theme Customization**
- **Astra Customizer:**
  - Header Builder for navigation
  - Footer Builder for site-wide footer
  - Color palette selection
  - Typography settings
  
- **Page Settings:**
  - Hide/Show page title
  - Sidebar configuration (No Sidebar)
  - Content width (Full Width / Stretched)
  - Transparent header toggle

### 3. **Content Management**
- **Posts vs Pages:**
  - **Pages:** Static content (Home, Mission) - "Set it and forget it"
  - **Posts:** Dynamic content (Products) - Can be categorized and tagged
  - **Media Library:** Centralized image storage, reusable across site

### 4. **Responsive Design**
- **Mobile-First Approach:**
  - Tested on mobile viewport
  - Responsive column stacking
  - Touch-friendly navigation
  - Optimized image sizes

### 5. **Navigation & Menus**
- **Menu Management:**
  - Primary navigation in header
  - Dropdown/submenu support
  - Mobile hamburger menu
  - Footer navigation links

### 6. **Performance Optimization**
- **Plugins Used:**
  - Starter Templates (for initial setup)
  - Spectra (for advanced blocks)
  - Minimal plugin usage for better performance

## Challenges & Solutions
